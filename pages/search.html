搜索

基于 SQLite FTS5 的即时搜索,覆盖文件名、标签与全文内容。

索引结构

每个位置的 .whale/index.db 中维护以下索引:

  • files_fts: 文件名 trigram 索引,支持模糊匹配。
  • fulltext_fts: 全文内容索引(对支持提取文本的文件)。
  • EXIF/元数据索引(对图片等媒体文件)。

搜索方式

  • 即时搜索: 输入即查,结果实时更新。
  • 标签搜索: 输入标签名快速筛选带该标签的文件。
  • 高级查询(SearchQuery): 10 个字段组合查询(文件名、标签、日期、评分等)。
  • 保存搜索: 常用条件可保存,下次一键复用。

搜索语法示例

# 搜索文件名包含 report 的文件
report

# 搜索带 5star 标签的文件
5star

# 搜索 workflow:done 的文件
done

# 组合:文件名含 invoice 且为 4 星以上
invoice 4star 5star
注意: 首次添加位置后,WhaleTag 会自动建立索引。后续新增/修改文件会增量更新。
