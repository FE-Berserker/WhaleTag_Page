WhaleTag 使用说明

本地优先、离线、隐私安全的文件管理与打标签桌面应用。

快速开始 GitHub 仓库

核心能力一览

📁

文件浏览

本地文件夹位置、只读标记、目录树、面包屑、虚拟滚动,支持 list/grid/gallery 等基础视图。
🏷️

标签系统

基于 .whale/wsd.json sidecar 聚合标签,支持互斥家族(评分/工作流/象限/智能日期/期间)。
🎛️

9 种视角

list、grid、gallery、Task(Kanban/Matrix/Gantt)、calendar、mapique、folderviz、tagcloud、knowledge-graph。
🔍

即时搜索

SQLite FTS5 索引文件名、标签与全文,支持高级查询与保存搜索。
🧩

17 个内置扩展

图片、视频、PDF、Office、ePub、字体、Draw.io、Excalidraw、Markdown 等 viewer/editor。
🤖

AI 助手

嵌入 Claude Code CLI,支持 ollama/openai HTTP provider,流式侧栏与只读护栏。

设计原则

  • 本地优先 / 离线: 无后端、无遥测、无强制云,所有数据都在本机。
  • 数据神圣: 任何 IO/删除操作 merge 优先于 wipe;错误必须上抛,绝不静默吞。
  • 安全默认: contextIsolation: true、nodeIntegration: false、sandbox: true。
  • 可移植元数据: 标签统一存 sidecar JSON,不改动文件名、不改变文件夹结构。
  • 平台一致: 路径三形态归一,支持 Mac/Linux/Windows。

开始使用

  1. 克隆仓库并安装依赖:
    npm install
  2. 启动开发模式:
    npm run dev
  3. 在 app 中添加本地文件夹位置,开始浏览与打标签。

详细步骤见 快速开始。
